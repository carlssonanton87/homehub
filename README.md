# HomeHub Lite

HomeHub Lite is a full-stack Django web application that helps users manage important home-related information in one secure place.

## Purpose
The purpose of this project is to provide users with a simple and intuitive way to store documents, contacts, and household expenses.

## Target Audience
- Home owners
- Renters
- Families managing household information

## Technologies Used
- Python
- Django
- HTML / CSS
- JavaScript
- Stripe (payments)
